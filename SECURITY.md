# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Sifero Cloud's cryptographic implementation, please report it responsibly.

**Email:** admin@sifero.cloud

**What to include:**
- Description of the vulnerability
- Steps to reproduce
- Potential impact assessment

**Response time:** We aim to respond within 48 hours.

**Disclosure policy:** We ask that you give us 90 days to fix the issue before public disclosure.

## Scope

This repository contains only the client-side cryptographic module. For vulnerabilities in the full Sifero Cloud service, please contact us at the email above.

## Bug Bounty

We currently do not have a formal bug bounty program, but we will credit all responsible disclosures publicly (with your permission).
